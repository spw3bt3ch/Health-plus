# Health Plus - Comprehensive Health Assessment Platform

A modern Flask web application for evaluating 14 key health metrics including BMI, cardiovascular health, stroke risk, metabolic health, respiratory health, and more.

## Features

- **14 Health Assessments:**
  - BMI (Body Mass Index)
  - Cardiovascular Health (Blood Pressure)
  - Stroke Risk Assessment
  - Metabolic Health (Non-Invasive)
  - Respiratory Health (Blood Oxygen Saturation)
  - Physical Fitness (Resting Heart Rate)
  - Body Composition (Body Fat Percentage)
  - Bone & Musculoskeletal (Posture Assessment)
  - Mental Health (PHQ-9 Depression Screening)
  - General Wellness (Body Temperature)
  - Functional Health (Grip Strength Test)
  - Lifestyle & Risk Screening
  - Vision (Visual Acuity Test)
  - Hearing (Pure-Tone Hearing Screening)

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and navigate to:
```
http://localhost:5000
```

## Usage

1. Navigate to the Assessments page to select a health metric to evaluate
2. Fill out the assessment form with your measurements
3. View results on the Results page
4. Complete multiple assessments to get a comprehensive health overview

## Technology Stack
